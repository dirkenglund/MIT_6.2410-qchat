Pre-lab exercises:¶

Do the following exercises before the laboratory.

  1. Understanding the physics of entangled photon generation by nonlinear optics. – Review the physics of spontaneous parametric down conversion [here] (https://en.wikipedia.org/wiki/5D_optical_data_storage). – solved exercise on entangled photon pairs produced by two type-1 SPDCs

  1. One-time-pad encryption exercise -solved exercise using jupyter interactive features.. go through text, cipher text, recovered text.. perhaps also include error correction

  1. Eve’s dilemma .. Alice and Bob should estimate the error.. now let Eve perform measurements on some fraction x of channel uses.. from our jupyter interactive exercise, see how Eve’s action changes the error rate.. and this should make clear that if the key size is smallish, then Eve’s action can go undetected

  1. Run the pre-lab experiment (Thorlabs experiment)

show how it works..

.. why is it not secure?

Laboratory Goals¶

  1. Obtain raw key of length X.

  • How many photons do you have to send to get the raw key?

  • How does changing the loss between Alice and Bob affect that? (Do we want them to add an attenuator to measure this?)

  1. Sift key

  • What is the fraction of sifted to raw?

  1. Error correction

  • What’s the error rate before EC? How does this change with key length?

  • What’s the highest efficieny of error code to get final BER < 1E-10?

  1. Privacy Amplification

  • What’s the higheest efficiency you can achieve for privacy amplification?

  • What’s the overall efficiency you were able to achieve from the distilled key?

  1. What about Eve?

  • How much did a passive Eve get in your experiment?

  • How would you build an experiment to measure that?

  • Now Eve has her own source. How can she get more information from the QKD setup.

  1. Overall SKR

  • What is the most time-consuming part of generating keys?

  • How would you speed that up?
